# CaféOS Demo - Video Creation Guide

## Quick Method: Use PowerPoint to Create MP4

### Step 1: Import Images
1. Open PowerPoint
2. Create 16 blank slides
3. Insert images in order:
   - Slide 1: `images/01-landing-hero.png`
   - Slide 2: `images/02-landing-features.png`
   - Slide 3: `images/03-landing-footer.png`
   - Slide 4: `images/04-login.png`
   - Slide 5: `images/05-dashboard.png`
   - Slide 6: `images/06-pos-menu.png`
   - Slide 7: `images/07-pos-cart.png`
   - Slide 8: `images/08-kitchen.png`
   - Slide 9: `images/09-affiliate-dashboard.png`
   - Slide 10: `images/10-affiliate-dashboard.png`
   - Slide 11: `images/11-affiliate-materials.png`
   - Slide 12: `images/12-affiliate-payouts.png`
   - Slide 13: `images/13-loyalty.png`
   - Slide 14: `images/14-superadmin-dashboard.png`
   - Slide 15: `images/15-superadmin-cafes.png`
   - Slide 16: `images/16-superadmin-settings.png`

### Step 2: Add Transitions
1. Select all slides (Ctrl+A)
2. Go to **Transitions** tab
3. Choose **Fade** (smooth)
4. Set duration: **4 seconds**
5. Check **"After"** and set to **4 seconds**

### Step 3: Add Captions (Optional)
Add text boxes with:
- Slide 1: "Welcome to CaféOS — The complete operating system for modern cafés"
- Slide 2: "Feature Overview — Powerful tools for every aspect of your business"
- Slide 3: "Ready to Transform — Get started with the future of café management"
- Slide 4: "Secure Authentication — Role-based access control"
- Slide 5: "Real-Time Dashboard — Monitor revenue and performance"
- Slide 6: "Point of Sale — Browse menu with lightning speed"
- Slide 7: "Shopping Cart — Review orders and checkout"
- Slide 8: "Kitchen Display System — Real-time order synchronization"
- Slide 9: "Affiliate Program — Earn commissions"
- Slide 10: "Affiliate Earnings — Track performance"
- Slide 11: "Marketing Materials — Professional promotional content"
- Slide 12: "Payout Management — Easy withdrawal system"
- Slide 13: "Loyalty Program — Reward customers"
- Slide 14: "Super Admin Dashboard — Platform-wide analytics"
- Slide 15: "Manage Cafés — Oversee all locations"
- Slide 16: "Platform Settings — Fine-tune CaféOS"

### Step 4: Export to MP4
1. Go to **File** → **Export** → **Create a Video**
2. Choose quality: **Full HD (1080p)**
3. Keep "Use Recorded Timings and Narrations"
4. Click **Create Video**
5. Save as `cafeos-demo.mp4`

**Done!** You'll have a professional 64-second MP4 video.

## Alternative: Online Tools (Even Faster)

### Kapwing (Free, No Download)
1. Go to https://www.kapwing.com/tools/slideshow-video-maker
2. Upload all 16 images from the `images/` folder
3. Set duration: 4 seconds per image
4. Add captions (copy from above)
5. Export as MP4

### Canva (Free)
1. Go to https://www.canva.com/
2. Choose "Video" template (1920x1080)
3. Upload all 16 images
4. Add them as pages (4 seconds each)
5. Download as MP4

## Pro Recording Method

If you want the EXACT demo with animations:
1. Open `index.html` in Chrome (full screen F11)
2. Press **Win + G** (Windows Game Bar)
3. Click **Record** button
4. Click **Play Demo** in the browser
5. Let it run for 64 seconds
6. Press **Win + Alt + R** to stop
7. Video saved in `Videos/Captures/` as MP4

**File size:** ~20-50 MB depending on quality
**Duration:** 64 seconds
**Resolution:** 1920x1080 (if browser is full screen)
